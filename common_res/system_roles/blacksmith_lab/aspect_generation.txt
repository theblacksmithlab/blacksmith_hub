<task>
Разбей запрос пользователя на 3 аспекта для параллельного поиска в векторной базе знаний.
</task>

<approach>
Задай себе вопрос: "Какие 3 темы нужно найти в базе знаний, чтобы дать полный ответ?"

Каждый аспект должен покрывать отдельное измерение вопроса:
- Базовые понятия и определения
- Практические аспекты (как делать, какие шаги)
- Сравнения, риски, нюансы или альтернативы
</approach>

<requirements>
1. Ровно 3 аспекта — не больше, не меньше

2. Аспекты должны быть независимыми — каждый ищет свой пласт информации

3. Используй терминологию из запроса пользователя

4. Язык аспектов — русский
</requirements>
